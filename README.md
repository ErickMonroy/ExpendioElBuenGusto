### Descripción
<div align=justify>

**Expendio "El Buen Gusto"** es un sistema web desarrollado para agilizar el procedimiento de compra en una cadena de sucursales dedicadas a la venta de pan tradicional y de temporada.  El sistema elimina las largas filas de espera permitiendo a los clientes realizar sus pedidos a través de pantallas de autoservicio, para posteriormente pasar únicamente a pagar y recoger su producto empaquetado.
</div>

### Objetivos principales:
- Reducir tiempos de espera en punto de venta
- Mejorar experiencia del cliente
- Digitalizar proceso de ordenamiento
- Generar tickets de compra automáticos

### Funcionalidades principales
- Registro de nuevos usuarios
- Inicio de sesión
- Busqueda de productos
- Selección de lista de productos
- Carrito de compras
- Agregar/remover productos
- Expedición de ticket de compra

## Tecnologías Utilizadas  
- **Frontend:** HTML5, CSS3, JS, SWeetAlert2, jsPDF
- **Almacenimiento de datos:** LocalStorage
- **Dependencias Externas (via CDN):** jQuery 3.6.0, Font Awesome 5.15.3, Google Fonts

## Requisitos del Sistema
- Navegador moderno (Chrome, Firefox, Edge)
- Conexión a Internet
- **JavaScript habilitado** en el navegador
